📌 行程總覽

4/22 - 4/26

東京5日悠閒美食之旅

放慢腳步，享受東京的每一個角落。沒有趕場的壓力，只有純粹的放鬆、美食與療癒的動物體驗。

📋 旅程摘要

  • 🗓️
    日期 4月22日 (三) - 4月26日 (日) 共計 5 天
  • 👥
    同行人數 3 人 (輕鬆舒適的小團體)
  • 🌟
    風格定調 步調悠閒、吃飽睡好、路線順暢不繞路。
  • 🎯
    必達成任務 吉伊卡哇樂園(17:00票)、任天堂旗艦店、秋葉原夾娃娃、銀座無印良品、動物園(明星獼猴 Punch)、猛男秀體驗。

🏨 住宿資訊

Dear cosmo 南池袋

📍 東京都豊島区南池袋1-14-6

入住時間 (Check-in) 4/22 15:00 後
退房時間 (Check-out) 4/26 10:00 前
✨ 住宿優勢

距離池袋車站步行約 5-8 分鐘。周邊全家、Lawson 齊全，採買宵夜極度方便。最重要的是，距離一級戰區「吉伊卡哇樂園 (池袋PARCO)」非常近，採購完回房放戰利品超省力！

📍 在 Google Maps 導航

經過重新編排後，這次行程的路線已經**極度順暢且省時**！請參考下方的每日轉乘指南。

🎟️ 必備票券建議

1交通 IC 卡 (Suica)

本次有許多 JR 與私鐵/公車混搭，強烈建議直接將 Suica 綁定 Apple Pay，或準備實體卡。全程約需儲值 ¥3,000 - ¥4,000。

2Skyliner 來回票

成田機場 ↔ 日暮里 最快速舒適。建議出發前先在 Klook/KKday 買好來回票，現場掃 QR Code 劃位最省時。

💡

溫馨提醒

因為會一直跨系統搭乘（JR、地鐵、公車），非常不建議購買「東京地鐵 24/48/72 小時券」，容易被限制路線反而繞遠路。一律無腦刷 Suica 最輕鬆！

🛤️ 每日詳細轉乘拆解

Day 1：機場 ➔ 池袋

成田機場 ➔ (搭乘 Skyliner 約 40 分) ➔ 日暮里站 ➔ (站內轉乘 JR山手線 約 13 分) ➔ 池袋站。

Day 2：池袋 ➔ 澀谷 ➔ 淺草 ➔ 池袋

上午出發：池袋 ➔ (JR埼京線/湘南新宿線，或副都心線直達) ➔ 澀谷。
下午轉戰：澀谷 ➔ (東京地鐵銀座線，起點坐到終點站，一線直達免換車！) ➔ 淺草 ➔ (地鐵/JR) ➔ 池袋。

Day 3：池袋 ➔ 築地 ➔ 銀座 ➔ 池袋

池袋 ➔ (東京地鐵丸之內線，於銀座轉日比谷線) ➔ 築地站 ➔ (步行) ➔ 銀座 ➔ (東京地鐵丸之內線) ➔ 池袋。

Day 4：池袋 ➔ 市川 ➔ 秋葉原 ➔ 新宿 ➔ 池袋

上午出城：池袋 ➔ (JR山手線轉總武線) ➔ 本八幡站 ➔ (北口搭乘京成巴士「本15」) ➔ 動植物園。
下午回城：本八幡站 ➔ (JR總武線一線直達！) ➔ 秋葉原 ➔ (JR中央線直達) ➔ 新宿 ➔ (JR) ➔ 池袋。

Day 5：池袋 ➔ 機場

池袋站 ➔ (JR山手線 約 13 分) ➔ 日暮里站 ➔ (轉乘 Skyliner 約 40 分) ➔ 成田機場。
